angular2-TvTracker

An Angular2/AngularFire2 example application.

Application Features

  • Angular 2.1
  • Modular feature design
  • Strongly typed models
  • Authentications guards
  • showAuth/hideAuth directives
  • Application data preloading
  • Heavy use of observables
  • Notification System

3rd Party Libraries

  • angularfire2
  • angular2-toaster
  • ng2-bootstrap

Firebase Features

  • AngularFire2
  • AngularFire2 authentication
  • Data paging via queries
  • Registration
  • Profile/email updates

All TV show data has been provided by the TVmaze API.
